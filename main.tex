\title{Learning Mealy Machines with Timers}
\author{Bengt Jonsson}
\affiliation{Department of Information Technology, Uppsala University}
\author{Frits Vaandrager}
\affiliation{Department of Software Science, ICIS, Radboud University, Nijmegen}

\begin{abstract}
We introduce a new model of Mealy machines with timers (MMTs) that is able to describe the timing behavior of a broad class of practical systems. We present two semantics for MMTs which, surprisingly, turn out to be equivalent: a timed trace semantics that records the times at which inputs and outputs occur, and an untimed trace semantics that records when and how timers are set and when they expire. We establish a Myhill-Nerode theorem for the untimed semantics and use this result to develop an active learning algorithm for MMTs following Angluin's approach of a minimally adequate teacher. 
\end{abstract}
